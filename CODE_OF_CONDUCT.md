# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and abusive
behavior are not tolerated. Report concerns to the maintainers via a GitHub issue
or, for sensitive matters, a private Security Advisory.

The full text is available at
https://www.contributor-covenant.org/version/2/1/code_of_conduct/.
